# 🚀 دليل تشغيل المشروع على سيرفرات Hetzner VPS (للمبتدئين)

هذا الدليل مصمم خصيصاً لمساعدتك في تثبيت وتشغيل أداة **Lofi Video Generator V3** على سيرفر (VPS) من شركة **Hetzner** من الصفر بدون أي خبرة سابقة في إدارة السيرفرات.

---

## 🏗️ المرحلة الأولى: شراء وإعداد السيرفر من Hetzner

1. قم بزيارة موقع [Hetzner Cloud](https://www.hetzner.com/cloud) وأنشئ حساباً.
2. بعد تسجيل الدخول، أنشئ مشروعاً جديداً (New Project).
3. اضغط على أضف سيرفر (Add Server).
4. **اختر المواصفات التالية:**
   - **Location (الموقع):** اختر أي موقع أوروبي (مثل: Falkenstein أو Nuremberg) ليكون السعر أرخص.
   - **Image (نظام التشغيل):** اختر **Ubuntu 22.04 LTS** (مهم جداً).
   - **Type (نوع السيرفر):** 
     - انقر على تبويب **Shared vCPU**.
     - لإنتاج فيديوهات 4 ساعات، ننصح **بشدة** باختيار الترقية كحد أدنى: **CPX31** (4 أنوية و 8 جيجا رام).
   - **Network:** اتركها (IPv4 & IPv6).
   - **Name (الاسم):** يمكنك تسميته `lofi-server`.
5. اضغط على **Create & Buy now**.
6. في غضون ثوانٍ، سيصلك إيميل من Hetzner يحتوي على عنوان الـ **IP للسيرفر** وكلمة المرور (Password) للمستخدم `root`.

---

## 🔌 المرحلة الثانية: الاتصال بالسيرفر للمرة الأولى

للدخول إلى السيرفر الخاص بك، استخدم برنامج اتصال آمن يسمى SSH.

### لمستخدمي Windows 10/11 أو Mac:
1. افتح موجه الأوامر (`Command Prompt` أو `Terminal`).
2. اكتب الأمر التالي مع تبديل الأصفار بعنوان الـ IP الذي وصلك في الإيميل:
   ```bash
   ssh root@0.0.0.0
   ```
3. إذا سألك `Are you sure you want to continue connecting?` اكتب `yes` واضغط Enter.
4. سيطلب منك كلمة المرور (انسخها من الإيميل. عند اللصق في الشاشة لن يظهر شيء من باب الأمان، فقط اضغط Enter).
5. (للمرة الأولى فقط) سيطلب منك السيرفر إدخال كلمة المرور القديمة (أدخلها مجدداً)، ثم كتابة كلمة مرور جديدة وتأكيدها.

أنت الآن داخل السيرفر! 🎉

## 🛠️ المرحلة الثالثة: تجهيز السيرفر (تثبيت الأدوات)

الآن ونحن داخل السيرفر، سنقوم بتنفيذ بعض الأوامر لتحديثه وتثبيت البرامج المطلوبة.
*(انسخ كل أمر من السطور التالية، الصقه في شاشتك السوداء، واضغط Enter)*

**1. تحديث النظام (قد يستغرق 3 دقائق):**
```bash
sudo apt update && sudo apt upgrade -y
```

**2. تثبيت بايثون وأدوات إدارة الملفات:**
```bash
sudo apt install python3 python3-pip python3-venv git unzip curl nano -y
```

**3. تثبيت أداة Muxer لدمج الفيديوهات (FFmpeg) - (الأساسية للمشروع):**
```bash
sudo apt install ffmpeg -y
```

**4. التأكد من نجاح تثبيت الأدوات:**
```bash
python3 --version
ffmpeg -version
```
*(إذا ظهرت لك أرقام الإصدارات فهذا يعني أن التركيب تم بنجاح).*

## 📥 المرحلة الرابعة: رفع المشروع إلى السيرفر وتشغيله

الآن سنقوم بنسخ مجلد مشروعك (الذي على GitHub) إلى السيرفر.

**1. استنساخ المشروع (Clone):**
*(بافتراض أنك رفعت مشروعك على GitHub، غير رابط المستودع إذا دعت الحاجة)*
```bash
git clone https://github.com/YourUsername/lofi-video-generator.git
```
*ملاحظة: إذا لم تكن قد رفعت مشروعك بعد لـ GitHub، يمكنك استخدام أدوات مثل FileZilla لنقله (باستخدام عنوان IP و root وكلمة المرور).*

**2. الدخول إلى مجلد المشروع:**
```bash
cd lofi-video-generator
```

**3. تثبيت حزم وإضافات بايثون المطلوبة:**
```bash
pip3 install -r requirements.txt
```

**4. تهيئة ملف الأسرار (المفاتيح):**
نحتاج إلى وضع مفاتيح واجهات برمجة التطبيقات (API Keys) في السيرفر.
انسخ ملف الأمثلة إلى ملف حقيقي مخفي `.env`:
```bash
cp .env.example .env
nano .env
```
ستفتح لك شاشة محرر النصوص:
- امسح الكلمات الوهمية والصق مفاتيحك بدلاً منها (مثل `GEMINI_API_KEY` و `KIE_API_KEY`).
- لحفظ الملف: اضغط على `Ctrl + X` ثم اضغط `Y` ثم `Enter`.

**5. إنشاء المجلدات المطلوبة مسبقاً لحفظ الفيديوهات:**
```bash
mkdir -p temp output
```

## 🎬 المرحلة الخامسة: البدء بتوليد فيديو 4 ساعات!

**الطريقة الأولى: من خلال سطر الأوامر المباشر (الأفضل للسيرفرات):**
```bash
python3 v3_automation_pipeline.py "قهوة الصباح" "https://example.com/image.jpg"
```
*(العملية ستستغرق ما يقارب 30 إلى 90 دقيقة لإنتاج فيديو 4 ساعات).*

**الطريقة الثانية: تشغيل واجهة الويب التفاعلية:**
إذا أردت استخدام الواجهة الزرقاء السهلة:
```bash
streamlit run web_interface.py
```
*(ستظهر لك رسالة تعتذر عن تشغيله أحياناً لأن السيرفر يحتاج إلى فتح المنفذ 8501، لاختبار الواجهة ابحث في جوجل عن "How to open port 8501 ubuntu ufw").*

## 🏃 تنويه هام جداً! (تشغيل السكربت وترك السيرفر)

بما أن صناعة الفيديو ستأخذ من 30 إلى 90 دقيقة، إذا أغلقت اللابتوب الخاص بك أثناء عمل السيرفر، سيتوقف المونتاج!
لحل هذه المشكلة نستخدم أداة تسمى **Tmux** (وهي تجعل السكربت يعمل حتى بعد إغلاق شاشتك):

1. لإنشاء نافذة خلفية تعمل دائماً:
   ```bash
   tmux new -s lofivideo
   ```
2. شغل فيها أمر المونتاج:
   ```bash
   python3 v3_automation_pipeline.py "رحلة فضائية"
   ```
3. للخروج بأمان مع ترك العملية مستمرة بالخلفية: اضغط باستمرار على `Ctrl + B`، اتركهما، ثم اضغط حرف `D`.
4. يمكنك إغلاق اللابتوب بكل أمان والعودة بعد ساعتين!
5. للعودة لاحقاً لرؤية ما حدث:
   ```bash
   tmux attach -t lofivideo
   ```

بعد الانتهاء ستجد فيديوهاتك بصيغة `.mp4` في مجلد الـ `output`. أفضل طريقة لسحب الفيديو من سيرفر Hetzner إلى جهازك هي عن طريق استخدام برنامج نقل الملفات مثل [FileZilla](https://filezilla-project.org/) باستخدام بروتوكول **SFTP** (نفس الـ IP واسم المستخدم root وكلمة المرور للسيرفر).
