### 2. 🔄 **عدم وجود Retry Logic كافي**

**الخطر:** متوسط

**المشكلة:**
- Gemini: عند نفاد Quota يتوقف فوراً
- Pexels: لا معالجة لـ Rate Limiting (429)
- Kie.ai: لا backoff عند الفشل

**التأثير:**
- ❌ فشل العملية بسبب أخطاء مؤقتة
- ❌ تجربة مستخدم سيئة
- ❌ هدر للـ API calls

---

### 3. 📝 **رسائل خطأ غير واضحة**

**المشكلة:**
```python
# مثال من Kie.ai
return {"success": False, "error": f"API Error {status_code}"}
```

**التأثير:**
- ❌ المستخدم لا يعرف ماذا يفعل
- ❌ صعوبة في الـ debugging
- ❌ لا توجيه واضح

**مثال محسّن:**
```python
if status_code == 402:
    return {
        "success": False,
        "error": "نفذ رصيدك من Kie.ai",
        "suggestion": "قم بشحن حسابك من: https://kie.ai/billing"
    }
```

## 🟡 **التحسينات المقترحة**

### 4. **Timeout غير مناسب للملفات الكبيرة**
- المشكلة: `timeout=60` ثابت
- الحل: استخدام `timeout=(connect, read)`

### 5. **عدم caching لنتائج Pexels**
- المشكلة: نفس البحث يُنفّذ مرات متعددة
- الحل: حفظ النتائج محلياً لمدة 24 ساعة

### 6. **Validation ضعيف لـ JSON من Gemini**
- المشكلة: يرجع `{}` فارغ بدون سبب
- الحل: طباعة الـ raw response للـ debugging

### 7. **YouTube 403 غير مفسرة**
- المشكلة: رسالة عامة فقط
- الحل: شرح الأسباب المحتملة + حلول

### 8. **Model names hardcoded**
- المشكلة: أسماء النماذج في الكود
- الحل: نقلها لـ config.py
