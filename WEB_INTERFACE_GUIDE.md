# 🎬 دليل الواجهة الجديدة - Lofi Studio

## 🌟 **المميزات الجديدة**

### ✅ **1. إدخال API Keys يدوياً**
- 🔑 لا حاجة للـ `.env` بعد الآن!
- إدخال مباشر من الموقع
- حفظ في الجلسة (آمن)

### ✅ **2. اختيار AI Provider**
- 🎯 **Auto** - يجرب الكل
- 🧠 **Gemini** - فقط
- 🌐 **OpenRouter** - فقط

### ✅ **3. تصميم احترافي جديد**
- 🎨 ألوان متدرجة جميلة
- 💫 تأثيرات hover
- 📱 responsive design
- 🌙 Dark theme فخم

### ✅ **4. Sidebar منظم**
- 4 أقسام (Tabs):
  - 🔑 API Keys
  - 🤖 AI Provider
  - 📺 YouTube
  - 📅 البث

---

## 🚀 **كيفية الاستخدام**

### **الخطوة 1: شغّل الواجهة**

```bash
streamlit run web_interface.py
```

---

### **الخطوة 2: أدخل API Keys**

#### **في Sidebar → 🔑 API Keys:**

1. **Gemini API (اختياري)**
   - افتح التبويب "🧠 Google Gemini AI"
   - الصق مفتاحك
   - ✅ سيظهر "متصل"

2. **OpenRouter API (اختياري - بديل لـ Gemini)**
   - افتح "🌐 OpenRouter AI"
   - الصق مفتاحك
   - نموذج مجاني: `openai/gpt-oss-120b:free`

3. **Kie.ai API (مطلوب! ⚠️)**
   - افتح "🎵 Kie.ai"
   - الصق مفتاحك
   - ❌ بدونه لن يعمل!

4. **Pexels API (اختياري)**
   - لجلب صور/فيديوهات مجانية

---

### **الخطوة 3: اختر AI Provider**

#### **في Sidebar → 🤖 AI Provider:**

اختر واحد:
- **🎯 تلقائي (Auto)** ← **موصى به!**
  - يجرب Gemini أولاً
  - إذا فشل → يجرب OpenRouter
  
- **🧠 Gemini فقط**
  - أفضل جودة
  
- **🌐 OpenRouter فقط**
  - مجاني بدون حدود

---

### **الخطوة 4: ابدأ الإنتاج!**

#### **في الصفحة الرئيسية:**

**المرحلة 1: التصميم (يسار)**
1. اكتب فكرة: "قهوة الصباح"
2. ارفع صورة
3. اضغط "🧠 توليد التفاصيل"
4. راجع وعدّل التفاصيل:
   - 🎵 وصف الموسيقى
   - 🎬 وصف التحريك
   - 📺 عنوان الفيديو
   - 📄 الوصف

**المرحلة 2: الإنتاج (يمين)**
5. اضغط "🎬 إطلاق الإنتاج الآن!"
6. انتظر (~5-10 دقائق)
7. ✅ الفيديو جاهز!
8. شاهد أو حمّل

## 💡 **حالات الاستخدام**

```
1. Sidebar → 🔑 API Keys
   - أدخل Gemini Key
   - أدخل Kie.ai Key

2. Sidebar → 🤖 AI Provider
   - اختر "🧠 Gemini فقط"

3. الصفحة الرئيسية
   - أدخل فكرة وصورة
   - اضغط توليد
```

---

### **حالة 2: عندك OpenRouter فقط**

```
1. Sidebar → 🔑 API Keys
   - أدخل OpenRouter Key
   - أدخل Kie.ai Key

   - اختر "🌐 OpenRouter فقط"

3. الصفحة الرئيسية
   - أدخل فكرة وصورة
   - اضغط توليد
```

---

### **حالة 3: عندك الاثنين (الأفضل!)** ⭐

```
1. Sidebar → 🔑 API Keys
   - أدخل Gemini Key
   - أدخل OpenRouter Key
   - أدخل Kie.ai Key

2. Sidebar → 🤖 AI Provider
   - اختر "🎯 تلقائي (Auto)"

3. الصفحة الرئيسية
   - أدخل فكرة وصورة
   - اضغط توليد
   - سيعمل دائماً! ✅
```

### **حالة 4: بدون AI (يدوي)**

```

   - أدخل فكرة وصورة
   - لا تضغط "توليد التفاصيل"
   - اكتب يدوياً:
     * وصف الموسيقى
     * وصف التحريك
     * العنوان والوصف
   - اضغط "إطلاق الإنتاج"
```

## ⚡ **الأداء**

### **السرعة:**
- تحميل الصفحة: < 1 ثانية
- Responsive: فوري
- ال API calls: حسب السيرفر

### **الذاكرة:**
- API Keys: في session_state
- لا تُحفظ في ملفات
- آمن 100%

---

## 🔐 **الأمان**

### **API Keys:**
- ✅ تُحفظ في st.session_state فقط
- ✅ لا تُكتب في ملفات
- ✅ type="password" (مخفية)
- ✅ تُحذف عند إغلاق المتصفح

### **البيانات:**
- الصور: في temp/uploads/
- الفيديوهات: في output/
- لا رفع لأي مكان خارجي
